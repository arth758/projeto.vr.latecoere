# Projeto-VR

-   Projeto para exibição de peças de avião em 3d

-   Principais dependências utilizadas:

    -   Front:
        -   Vue JS (UI)
        -   vue-3d-model (Exibição das peças gltf)
    -   Back:
        -   Express (API)
    -   axios (Comunicação Front-Back)

-   Para iniciar a aplicação:
    -   Front:
        ```
        cd client
        npm run serve
        ```
    -   Back:
        ```
        cd server
        npm start
        ```
